# Lightwallet Protocol Status

Lightwallet protocol infrastructure is out of scope for BerzCoin v0.1.

Current status:
- Dedicated lightwallet/stratum surfaces are removed or deprecated.
- The supported model is full-node validation plus private-key wallet activation.

If a lightwallet protocol is reintroduced in a later version, this document should be replaced with:
- protocol goals
- trust/security model
- wire messages
- server indexing assumptions
- compatibility guarantees
